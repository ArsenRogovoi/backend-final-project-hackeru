# Back end part of final project of HackerU bootcamp

## "New Visit" server:

### About:

This is REST API server for application "New Visit" (To client git hub repo [click here](https://github.com/ArsenRogovoi/frontend-final-project-hackeru)). The server responsible to do CRUD operations for users and appointments, authorization and managing data with MongoDB.

### Installation:

1. Make sure you have MongoDB installed locally and it server runs when you use this backend.
2. Download this repo. You can do this in console with git command:

```bash
git clone https://github.com/ArsenRogovoi/backend-final-project-hackeru.git
```

3. Install all the dependencies of the app:

```bash
npm i
```

4. To run the server use:

```bash
npm run start
```
